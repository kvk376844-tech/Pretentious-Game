# Pretentious Game

## Project Overview

Pretentious Game is an avant-garde indie adventure where players embody a self-absorbed philosopher navigating a surreal world of existential absurdities. Set in a minimalist, black-and-white landscape inspired by pretentious art films, you ponder life's meaningless questions while solving puzzles that mock intellectualism—think rearranging floating coffee cups to symbolize "the burden of choice" or debating with AI-generated NPCs about the ethics of pixelated existence. With no combat, just endless monologues and ironic meta-commentary, the game culminates in a "profound" ending that reveals it's all a simulation of your own ego. Built on Pygame for cross-platform pretension, it features procedurally generated poetry and a soundtrack of ambient sighs. Perfect for players who want to feel smarter without actually learning anything.

## Key Features
- **Existential Puzzles**: Interact with objects that represent deep philosophical concepts (or so they claim).
- **Meta-Narrative**: Break the fourth wall with self-referential dialogue.
- **Procedural Poetry**: Generate haikus on the fly about the futility of gaming.
- **Minimalist Art Style**: Black-and-white visuals to emphasize "artistic depth."

## Getting Started
Clone the repository and run the main script with Python (version 3.8 or later). Ensure you have Pygame installed via `pip install pygame`.

For a code example, see `main.py`, which demonstrates the Philosopher Monologue Generator. This could be integrated into a Pygame scene for NPC interactions.
